## 1. 議員データ

### 1.1 参議院議員（248名）

| 項目 | 内容 |
|------|------|
| **ファイル** | `packages/ingestion/src/politicians/seed-councillors.ts` |
| **データ件数** | 248名（現員247名＋欠員1） |
| **基準日** | 令和8年（2026年）2月12日現在 |
| **一次ソース** | [参議院公式 会派別所属議員名一覧](https://www.sangiin.go.jp/japanese/joho1/kousei/giin/220/giinmei.htm) |
| **補助ソース** | [参議院公式 選挙区・比例代表別議員一覧](https://www.sangiin.go.jp/japanese/giin/hireiku/hireiku.htm) |

**収集方法**: 参議院公式サイトの会派別議員一覧ページを参照し、全12会派の所属議員を手動で転記。選挙区・比例代表の区分は参議院公式の選挙区別一覧と照合。

**会派別内訳**:
- 自由民主党・無所属の会: 101名（うち無所属3名: 寺田静, 出川桃子, 宮本和宏）
- 立憲民主・無所属: 40名
- 国民民主党・新緑風会: 25名
- 公明党: 21名
- 日本維新の会: 19名
- 参政党: 15名
- 日本共産党: 7名
- れいわ新選組: 5名
- 沖縄の風: 2名
- チームみらい・無所属の会: 2名
- 社会民主党: 2名
- 日本保守党: 2名
- 各派に属しない議員: 6名

**制約・注意点**:
- 会派と政党は異なる場合がある（例: 無所属だが自民会派に所属）
- 合区選挙区（鳥取県・島根県、徳島県・高知県）は都道府県を `null` に設定
- 第26回（2022年）改選組＋第27回（2025年）改選組の合計

## 3. 政治資金データ

### 3.1 実データ版（client.ts）

| 項目 | 内容 |
|------|------|
| **ファイル** | `packages/ingestion/src/political-finance/client.ts` |
| **データ件数** | 10政党×3年（2021-2023）＝30報告書＋収支明細 |
| **一次ソース** | [総務省 政治資金収支報告書](https://www.soumu.go.jp/senkyo/seiji_s/seijishikin/) |

**補助ソース・照合先**:
- [nippon.com 政党交付金データ](https://www.nippon.com/ja/japan-data/h01393/) — 2022年確定額
- [nippon.com 2024年政党交付金](https://www.nippon.com/ja/japan-data/h01961/) — 2024年確定額
- 日本経済新聞「自民収入244億円で首位 19年」（2020/11/28）
- 日本経済新聞「自民党、23年収入は439億円」（2024/3/17）
- しんぶん赤旗「自民収入 7割が税金/21年政治資金収支報告書」（2022/11/26）
- [日本共産党 2022年収支報告](https://www.jcp.or.jp/akahata/aik23/2023-11-25/2023112505_01_0.html)

**収集方法**: 総務省が毎年11月に公表するPDF版の政治資金収支報告書から主要数値を抽出。政党交付金額は総務省の確定額通知に基づく正確な値。収入内訳（党費・個人献金・企業献金・事業収入等）は報告書の分類に従う。

**精度について**:
- **政党交付金**: 総務省の確定通知に基づく正確な値
- **総収入・総支出**: 公表された報告書の数値（億円単位で正確）
- **収入内訳**: 報告書の分類カテゴリに基づく。端数は `OTHER_INCOME` に集約
- **日本共産党の特殊性**: 政党交付金0円、企業・団体献金0円を正確に反映。収入の約87%がしんぶん赤旗等の事業収入

- 第217回: 能動的サイバー防御法、AI推進法、年金改革法

**法案ステータスのマッピング**:
| DB値 | 意味 | 件数 |

| ENACTED | 成立 | 約80件 |
| REJECTED | 否決 | 2件 |

| SUBMITTED | 提出 | 数件 |

## データ品質に関する注意事項

### 正確性のレベル

| レベル | 説明 | 該当データ |
|--------|------|-----------|
| **A（公式確定値）** | 政府機関の公式発表値 | 選挙結果（投票率・議席数）、政党交付金額、国会会期 |
| **B（公式報告書準拠）** | 公式報告書から転記した値 | 政治資金収支、法案の審議状況 |
| **C（報道ベース）** | メディア報道に基づく値 | 収入内訳の一部、政策比較 |

### 更新頻度

- **議員データ**: 選挙後・補欠選挙後に更新が必要
- **政治資金データ**: 毎年11月の総務省公表後に更新
- **法案データ**: 各国会会期終了後に更新
- **選挙データ**: 選挙実施後に更新

### 既知の制約

1. **衆議院比例復活当選者**: 一部の比例復活当選者の確認が困難なケースあり（動的Webページのため機械的収集が難しい）
2. **政治資金の内訳**: 総務省のPDFからの転記のため、端数処理により合計値と内訳の和に若干の差異が生じる場合がある
3. **会派異動**: 議員の会派異動は反映が遅れる場合がある
4. **過去データの遡及**: 2013年以前の選挙データは未収録
